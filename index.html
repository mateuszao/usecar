CARSHARING UseCar

O sistema UseCar Carsharing é uma solução completa em recursos. O modelo de negócio oferece tanto a tecnologia para compartilhar e liberar o funcionamento dos veículos por aplicativo, quanto o fornecimento e manutenção dos veículos. A plataforma web que é acessível para diferentes dispositivos como computadores, smartphones e tablets. É uma plataforma desenvolvida para suportar a utilização de milhares de usuários simultâneos, com rapidez e eficiência.
